# 📊 Sales Dashboard (Power BI)

## 🔍 Overview

This project is a Power BI dashboard analyzing sales performance, profit trends, and customer insights.

## 🚀 Features

* KPI Cards (Sales, Profit, Orders)
* Dynamic Top N Analysis
* Sales Growth Tracking
* Region-wise Sales Map
* Customer & Product Insights
* Drillthrough and Interactive Filters

## 🛠 Tools Used

* Power BI
* DAX
* Data Modeling

## 📈 Key Insights

* Technology category contributes highest revenue
* Some products have high sales but low profit
* Regional performance varies significantly
